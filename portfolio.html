My experience with Web Design


Before this course, I was worried that I might not be able to code a website properly. I did some coding for games before, but this website coding is on another level!

Coding a website isn't that hard to be honest. You can copy paste the codes with ease, but you need to know how they function so you can code properly. My first websites were not really polished, but for a beginner I was truly amazed by the results. Who knew coding websites could be so much fun?

I started with the most basic website ever. Then I just learn more codes and improve my skills, mostly thanks to my teacher Shruthi. I never tried to hide my confusion when my codes didn't work, and she has fully supported me and my class when needed.


After publishing two websites, I feel great now that I can actually do coding. I will try to code more websites for my personal projects in the future.
